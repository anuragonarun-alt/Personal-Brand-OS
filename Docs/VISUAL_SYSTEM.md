# Personal Brand OS — Visual System

## Visual Direction

References indicate a blend of:

* OpenCode
* Polymarket
* Dataism
* Linear
* Research dashboards
* Information design systems

The goal is NOT futuristic sci-fi.

The goal is intelligent minimalism.

# Cards

Background:
#0D0D0D

Border:
1px solid #1A1A1A

Radius:
12px

Shadow:
None

Cards should feel attached to the system.

Not floating.

---

# Borders

Borders are important.

Use borders more than shadows.

Use separators frequently.

Visual structure should come from grids and borders.

---

# Information Density

Good:

* Compact metric cards
* Multiple panels visible
* Dense dashboards
* Fast scanning

Bad:

* Giant cards
* Empty whitespace
* Marketing-page layouts

---

# Charts

Charts should be minimal.

Use:

* Line charts
* Heatmaps
* Activity grids
* Timelines
* Trend indicators
* Signal bars

Avoid:

* Pie charts
* Bright colors
* Decorative illustrations

---

# Visual Motifs

Allowed:

* Dot matrices
* Signal indicators
* Activity heatmaps
* Data grids
* Network structures
* Growth timelines
* System status panels

Use sparingly.

They support the interface.

They are not decorations.

---

# Motion

Allowed:

* Fade
* Slide
* Number counting
* Hover transitions

Avoid:

* Glows
* Particle effects
* Floating animations
* Excessive motion

---

# Things To Avoid

Do NOT generate:

* Glassmorphism
* Neon cyberpunk
* Hacker aesthetics
* Matrix effects
* Heavy gradients
* Bright colorful dashboards
* Floating cards with shadows
* Rounded blob shapes
